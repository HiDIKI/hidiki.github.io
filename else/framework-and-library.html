# * 프레임워크와 라이브러리의 차이

Written by 📝 Hidekuma

주도권을 누가 가지고 있냐로 프레임워크와 라이브러리를 구분한다.
특징 프레임워크 라이브러리
유저코드의 작성 프레임워크 클래스를 서브 클래스로 작성 독립적으로 작성
호출흐름 프레임워크가 유저코드를 호출 유저코드가 라이브러리를 호출
실행흐름 프레임워크가 제어 유저코드가 제어
객체의 연동 프레임워크가 정의 독자적으로 정의
